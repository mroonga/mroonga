# News - 16 series

(release-16-01)=
## Release 16.01 - 2026-03-30

### Improvements

### Added support for Percona Server for MySQL 8.0.45-36 in the following OS

* AlmaLinux 8, 9

8.0.45-36 is the latest Percona Server for MySQL 8.0 release.

### Added support for Percona Server for MySQL 8.4.8-8 in the following OS

* AlmaLinux 9, 10

8.4.8-8 is the latest Percona Server for MySQL 8.4 release.

### Added support for MariaDB 11.8.6 on Debian 13

This is the latest MariaDB 11.8 release.

### Dropped support for `engine "XXX"` in CONNECTION for wrapper mode

[[MDEV-38530]( https://jira.mariadb.org/browse/MDEV-38530 )][Reported by Roel Van de Paar]

Mroonga accepted `engine "XXX"` for wrapper mode in not only in `COMMENT` but also in `CONNECTION`.
However, this behavior was neither documented nor tested, so support for it has been removed.

### Fixes

#### Fixed a crash bug that occurred when `mroonga_log_file` was set `NULL` or an empty string

[[GH-1069]( https://github.com/mroonga/mroonga/pull/1069 )][Patched by Hadeer Ramadan]

Setting `NULL` or an empty string as below, caused a `SIGSEGV`.

```sql
SET GLOBAL mroonga_log_file = NULL;
```

```sql
SET GLOBAL mroonga_log_file = '';
```
